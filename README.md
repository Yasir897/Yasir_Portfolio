# Yasir's Portfolio

My personal portfolio website, built with **Django** + HTML/CSS/JavaScript. Bio,
education, skills, experience and projects are managed through the database and
rendered with Django templates.

## Tech stack
- Django (Python)
- HTML / CSS / JavaScript
- SQLite

## Run locally
```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
Then open http://127.0.0.1:8000

See `DEPLOY.md` for deployment instructions.
